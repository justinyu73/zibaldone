# Zibaldone v0.8.6

## 這版重點

- 修正 macOS 更新後遺留的舊版 Python sidecar 佔用 `127.0.0.1:8766`，造成新版本後端無法啟動的問題。
- 僅在啟動命令與 `/api/health` 回應都符合 Zibaldone 指紋時，才回收舊版 sidecar；不會誤殺其他 Python 或 HTTP 程序。
- 後端成本摘要無法讀取時，介面改為明確顯示「狀態未知」，不再錯誤顯示額度內。
- Windows 與 macOS release workflow 均會 smoke test 實際打包的 sidecar，包含健康檢查與受保護成本 API。

## 驗證

- Rust sidecar lifecycle tests：6/6 passed。
- Bundled sidecar smoke：health 與 protected cost summary passed。
- Frontend unit tests：48/48 passed；設定頁 Playwright：4/4 passed。
- Windows 與 macOS 安裝包由 `v0.8.6` release workflow 在原生 GitHub runner 產生。
