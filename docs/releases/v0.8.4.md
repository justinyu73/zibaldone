# Zibaldone v0.8.4

## 這版重點

- 修正 macOS Apple Silicon 發行包大小 gate：保留 Windows 110 MB 硬上限，macOS 改為 200 MB 硬上限，避免把不同平台的壓縮特性誤判為建置失敗。
- macOS 本版預期約 177–182 MB，Windows 約 96 MB；這是因為安裝包包含本機 FastAPI sidecar，不包含 Gemma 模型。
- 其餘功能與 v0.8.3 相同，包括 CLI 勾選後明確提示必須儲存、無金鑰時可產生待人工整理草稿，以及 Windows 不跳出終端機視窗。

## 下載

- Windows x64：`Zibaldone_0.8.4_x64-setup.exe`
- macOS Apple Silicon：`Zibaldone_0.8.4_aarch64.dmg`

請從本 Release 下載並核對對應的 `SHA256SUMS-*.txt`。Windows 未使用 Authenticode，macOS 未 notarize；SmartScreen／Gatekeeper 警告仍是預期風險。
