# Zibaldone v0.8.3

## 可下載版本

- Windows x64：`Zibaldone_0.8.3_x64-setup.exe`
- macOS Apple Silicon：`Zibaldone_0.8.3_aarch64.dmg`

## 這版重點

- CLI 模型開關勾選後，直接顯示「設定尚未儲存：CLI 尚未生效」，並在儲存按鈕旁顯示未儲存狀態。
- 無字幕影片可走本機 ASR/OCR，沒有 AI 金鑰時仍可建立可編輯的待人工整理草稿。
- Windows 發行版不再於每次啟動跳出額外終端機視窗。
- 內建本機 AI 與 ffmpeg 採首次使用下載；首次設定可跳過，不會阻塞完成設定。

## 已驗證

- 公開測試版完整 Playwright：27 passed。
- 發版 workflow 會先執行 backend、frontend、Playwright、Rust 與產品完整性 gate，通過後才建置 Windows/macOS 安裝檔。

## 安裝風險

此版本目前沒有 Windows Authenticode 或 Apple notarization，因此 SmartScreen／
Gatekeeper 警告是預期風險。請只從本 Release 下載，核對 `SHA256SUMS-*.txt`，
再依 [Windows](../install/windows.md) 或 [macOS](../install/macos.md) 指南處理，
不要關閉整台電腦的安全防護。
